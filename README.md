## 💡 Troubleshooting common issues

Most users find the process simple, but issues happen. Use this list to solve common problems.

**The application runs slow.**
Your computer might have many files to process. Check if you have other heavy programs open. Close them to give PixelRAG more power.

**The search yields no results.**
Check if the selected folder contains the files you need. Ensure your search phrase matches the text or visual elements in your documents.

**The installation stops midway.**
Check your internet connection. A stable connection ensures the download completes without errors. If the issue persists, run the installer as an administrator by right-clicking the file and selecting "Run as administrator."

**The software does not display images.**
Ensure your firewall allows PixelRAG to access local storage. Some security software blocks new programs from reading files. Grant permissions in your security settings to fix this.
